Bienvenidos al módulo de Lenguajes de Marcas

Los contenidos de este ejercicio tratan de ilustrar el funcionamiento de los elementos básicos de un documento HTML

Encabezados

Los encabezados tienen como función estructurar el contenido del documento en partes para identificar los apartados del texto.
Normalmente, se utiliza un encabezado nivel 1, que corresponde con el de mayor tamaño, para identificar los apartados principales.
Sepueden ir utilizando otros encabezados de menos tamaño para indicar los subapartados. Existen 6 tipos de encavezados.

Lineas horizontales

Suelen utilizarse las lineas horizontales para introducir una separación visual en el texto del documento.
